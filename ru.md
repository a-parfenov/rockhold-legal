# Юридические документы RockHold

RockHold — local-first приложение для учёта личных финансов на iOS. Production cloud MVP поддерживает аккаунт RockHold, вход через показанных поставщиков и синхронизацию поддерживаемых данных с сервером RockHold.

<div class="notice">
  Текущий статус: кандидат production cloud MVP. Полное удаление уже доступно по privacy email; до внешнего распространения должны быть добавлены in-app инициация, срок обработки, retention-исключения и полные реквизиты оператора.
</div>
