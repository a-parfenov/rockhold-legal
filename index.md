# RockHold Legal Documents

RockHold is a local-first personal-finance app for iOS. The production cloud MVP supports a RockHold account, authentication through displayed providers, and synchronization of supported data with the RockHold server.

  Current status: production cloud MVP candidate. Complete deletion is already available through the privacy email; in-app initiation, processing time, retention exceptions, and complete operator details must be completed before external distribution.
</div>
